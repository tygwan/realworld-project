# Assets

This directory describes local and generated assets used by the project.

Payloads are ignored by default when they are private or large. Track asset
metadata in `docs/reference/assets/ASSET-REGISTRY.md`.

Suggested subdirectories:

```text
assets/user-provided/
assets/private/
assets/generated/
assets/public/
```

Only commit assets when their license, privacy boundary, and file size are
explicitly acceptable.
