# User-Provided Assets

Local-only user-provided files can be placed here if convenient.

Examples:

```text
assets/user-provided/refinery/refinery_model.glb
assets/user-provided/refinery/installation_schedule.csv
```

These files are ignored by git by default. Record their role and local path in
`.env` and `docs/reference/assets/ASSET-REGISTRY.md`.
