# Documentation Index

This directory contains the project documentation for `realworld-project`.
It follows `dev-standards` R1: documentation architecture.

## Entry Point

Start with [PROJECT-JOURNAL.md](PROJECT-JOURNAL.md). It is the single portal for
project state, decisions, dependencies, open questions, and links to detailed
documents.

## Update Rules

- Add a decision record when a structural choice is made.
- Add a task log when a phase or significant task is completed.
- Add a finding archive when a technical/data issue is discovered.
- Keep the project journal current enough that a new collaborator can orient
  from that file alone.
