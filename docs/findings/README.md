# Findings

Archived issues live here. Use this directory for technical, data, dependency,
or design problems that require evidence and later follow-up.

Each finding should include:

- severity and status
- what was discovered
- reproducible evidence
- root cause analysis
- options considered
- selected resolution or deferral
